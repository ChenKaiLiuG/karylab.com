若你想**透過 Ubuntu 的 CLI（Command-Line Interface）查詢硬碟使用量**，你可以使用以下幾個指令。這些指令各有不同的用途，可以根據你的需求選擇最適合的： `df` 指令 (disk free) `df` 指令是最常用來查詢硬碟使用量的工具，它會顯示所有檔案系統的容量、已用空間、可用空間和掛載點。 ```Bash df -h ``` `-h`：這是 --human-readable 的縮寫。加上這個參數，df 會以更容易閱讀的方式顯示容量，例如 G (Gigabytes)、M (Megabytes) 或 T (Terabytes)，而不是預設的位元組（bytes），這樣會清楚許多。 執行後，你會看到類似下面的結果： 檔案系統 容量 已用 可用 已用% 掛載點 udev 1.9G 0 1.9G 0% /dev tmpfs 385M 1.6M 384M 1% /run /dev/sda2 45G 12G 33G 27% / tmpfs 1.9G 0 1.9G 0% /dev/shm tmpfs 5.0M 0 5.0M 0% /run/lock tmpfs 1.9G 0 1.9G 0% /sys/fs/cgroup /dev/sda1 511M 3.5M 508M 1% /boot/efi tmpfs 385M 16K 385M 1% /run/user/1000 在這個表格中，你可以看到每個檔案系統的詳細資訊。通常，你最需要關注的是掛載點在 / 的那一列，這代表你的根目錄，也就是系統的主硬碟分割區。 du 指令 (disk usage) du 指令是用來查詢 特定目錄 或 檔案 所佔用的空間，而不是整個硬碟。如果你想知道某個資料夾到底佔用了多少空間，du 是最佳選擇。 若要查詢 當前目錄 的總使用量，你可以使用： `Bash du -sh . -s：這是 --summarize 的縮寫，表示只顯示總計。 -h：--human-readable，以人性化格式顯示。 .：代表當前目錄。 如果你想知道 當前目錄下所有子目錄 的使用量，但不包含單一檔案，可以使用： Bash du -h --max-depth=1 --max-depth=1：限制只顯示第一層子目錄。 總結 使用 df -h 來快速查看整個系統的硬碟使用概況。 使用 du -sh [目錄路徑] 來查詢特定資料夾的大小。 這兩個指令基本上能滿足你在 Ubuntu 上查詢硬碟使用量的所有需求。
